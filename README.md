# 🏀 BasketPro - Site sobre Basquete

Um site moderno, responsivo e interativo dedicado ao mundo do basquete!

## 📋 Sobre o Projeto

O **BasketPro** é um portal completo que oferece informações sobre:
- 📜 História e origens do basquete
- 🌍 Alcance global do esporte
- 🏆 Principais competições (NBA, EuroBasket, Olimpíadas, NBB)
- 📚 Regras básicas do jogo
- 🎯 Curiosidades interessantes
- 💬 Formulário de contato

## ✨ Recursos

### Design Responsivo
- Layout totalmente responsivo que se adapta a qualquer tamanho de tela
- Otimizado para dispositivos móveis, tablets e desktops
- Grade CSS moderna com Flexbox e Grid

### Interatividade
- Navegação suave entre seções
- Animações fluidas ao carregar a página
- Validação de formulário
- Efeitos hover interativos
- IntersectionObserver para animações de entrada

### Cores Vibrantes
- Paleta de cores inspirada no basquete
- Gradientes modernos
- Contraste apropriado para acessibilidade

## 🚀 Como Usar

1. **Clone ou baixe o repositório:**
```bash
git clone https://github.com/nilmarmuller-coder/basquete.git
cd basquete
```

2. **Abra o arquivo `index.html` no navegador:**
- Simplesmente clique duas vezes no arquivo `index.html`
- Ou arraste para seu navegador favorito
- Ou use um servidor local (veja abaixo)

3. **(Opcional) Use um servidor local:**
```bash
# Com Python 3
python -m http.server 8000

# Com Node.js (se tiver http-server instalado)
http-server

# Com Live Server do VS Code
# Instale a extensão e clique em "Go Live"
```

Depois acesse: `http://localhost:8000`
